What is a design-build-test tool?

It is a tool for automating all design decisions in a design-build-test cycle

Automated design decomposition

Phoenix uses grammars to validate incoming design specifications and decompose them into functional building blocks for testing.

Data standards

Phoenix complies to existing data standards in synthetic biology. Phoenix supports SBOL and SBOL visual.

Automated experimental designs

The primary mechanism with users is set of cloning and testing instructions. These instruction sets are created from design decompositions and DNA assembly tools and are the results are interpreted for simulation to inform the next round of building and testing.
